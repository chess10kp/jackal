=== What Jackal is ===

A Jac-native terminal coding agent. Launch with ./jackal.sh from any project directory.

Stack:
- Ink TUI: templates/shell.cl.jac (compiled via jac-ink)
- Headless runtime: src/ (adapter, store, bridge, auth, MCP client)
- System prompt: pi/SYSTEM.md
- MCP config: pi/mcp.json (jac mcp server)
- Skills: pi/skills/*/SKILL.md (on demand)
- Project config: .jackal (per-project JSON)

The legacy Pi extension was removed. Launch only via ./jackal.sh (see docs/CONSOLIDATION_PLAN.md).

=== Launch ===

From repo root (or via symlink in PATH):

  cd /path/to/your/jac-project
  /path/to/jackal/jackal.sh

First-time setup from the jackal repo:

  npm install
  npm run build:agent
  ./jackal.sh

=== Shell slash commands (Ink TUI) ===

Implemented in templates/shell.cl.jac and/or src/runtime/tools.ts:

  /help          Command reference
  /exit, /quit   Quit
  /clear, /new   New session
  /abort         Cancel active agent run
  /multiline     Toggle multiline input
  /compact       Compact conversation context
  /login         Provider auth flow
  /logout        Logout provider
  /model         Model picker
  /jac-doctor    Environment + Jac toolchain check
  /jac-check     Run jac check diagnostics
  /fix           Fix loop (validate + patch, capped retries)
  /create        jac create template wrapper
  /jac <args>    Pass-through to jac CLI

Still to port into the Jackal runtime: full plan-mode step UI, LSP tools, mermaid rendering.
See docs/FEATURES.md for status.
