# Phase P3 — Long Sessions

**Exit criteria:** `/resume` loads prior session; auto-compact at threshold; `/usage` shows context %.

## Runtime (implemented)

| Module | Role |
|--------|------|
| `src/runtime/session-index.ts` | `listSessions`, `loadSessionById`, `resolveSessionTarget` (`last`, index, id) |
| `src/runtime/session.ts` | Per-session JSON + index; 30s auto-save; compaction backup slot |
| `src/runtime/context-usage.ts` | Token estimate (chars/4); max from model or override |
| `src/runtime/agent-session.ts` | `compactContext({ preview, restore })`, `resumeFromRecord`, `getContextUsage` |
| `src/adapter.ts` | `listSessions`, `resumeSession`, `exportSessionMarkdown`, `setContextMax`, `compactSession(opts)` |

Storage layout:

```
.jackal/sessions/
  index.json
  sess_<ts>.json
  sess_<ts>/compaction-backup.json
```

## Slash handlers — shell.cl.jac binding

Wire in `templates/shell.cl.jac` `submit_main()` (facade methods on `session` from `useJackalSession()`):

| Command | Facade / adapter call | Notes |
|---------|----------------------|-------|
| `/resume` | `session.listSessions()` → show overlay | **TODO:** Ink session picker component |
| `/resume last` | `session.resumeSession("last")` | Ready |
| `/resume {id}` | `session.resumeSession(id)` | Ready |
| `/resume {n}` | `session.resumeSession(String(n))` | 1-based index |
| `/rename {name}` | `session.renameSession(name)` | Ready |
| `/export` | `session.exportSessionMarkdown()` → write file or notify | **TODO:** pick path / clipboard |
| `/compact` | `session.compactSession({})` | Ready |
| `/compact --preview` | `session.compactSession({ preview: true })` | Shows summary via notify |
| `/compact --restore` | `session.compactSession({ restore: true })` | Restores backup |
| `/usage` | `session.getUsageLine()` or `getContextUsage()` | **TODO:** statusbar widget |

CLI flag (facade boot): `./jackal.sh --context-max 200000`

Config (`.jackal` JSON):

```json
{
  "contextMax": 128000,
  "sessions": {
    "autoSave": true,
    "saveIntervalMs": 30000
  }
}
```

Env: `JACKAL_CONTEXT_MAX=128000`
