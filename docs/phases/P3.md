## Runtime (implemented)

| Module | Role |
|--------|------|
| `src/runtime/session-index.ts` | `listSessions`, `loadSessionById`, `resolveSessionTarget` (`last`, index, id) |

| `src/runtime/context-usage.ts` | Token estimate (chars/4); max from model or override |

## Slash handlers — shell.cl.jac binding

Wire in `templates/shell.cl.jac` `submit_main()` (facade methods on `session` from `useJackalSession()`):

| Command | Facade / adapter call | Notes |
|---------|----------------------|-------|
| `/resume` | `session.listSessions()` → Ink `ResumePickerOverlay` | **done** |
| `/resume last` | `session.resumeSession("last")` | Ready |
| `/resume {id}` | `session.resumeSession(id)` | Ready |
| `/resume {n}` | `session.resumeSession(String(n))` | 1-based index |
| `/rename {name}` | `session.renameSession(name)` | Ready |
| `/export` | `session.exportSessionToFile()` → `.jackal/exports/` | **done** |
| `/compact` | `session.compactSession({})` — LLM default, mechanical fallback | **done** |
| `/compact --preview` | `session.compactSession({ preview: true })` | **done** |
| `/compact --restore` | `session.compactSession({ restore: true })` | **done** |
| `/compact --llm` | Force LLM summarization | **done** |
| `/compact --mechanical` | Mechanical compaction only | **done** |
| `/usage` | Toggle `UsageDetailOverlay`; status bar shows progress bar + % | **done** |
| `/context-max {n}` | `session.setContextMax(n)` | Ready |
| `/context-max` | `session.getContextUsage().max` | Show current max |

CLI flag (facade boot): `./jackal.sh --context-max 200000`

Config (`.jackal` JSON):

```json
{
  "contextMax": 128000,
  "compactStrategy": "llm",
  "autoCompact": {
    "enabled": true,
    "thresholdPercent": 80,
    "strategy": "llm"
  },
  "sessions": {
    "autoSave": true,
    "saveIntervalMs": 30000,
    "maxCount": 50,
    "retentionDays": 30
  }
}
```

Env: `JACKAL_CONTEXT_MAX=128000`
